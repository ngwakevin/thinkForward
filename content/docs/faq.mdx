# FAQ

### How opinionated is the guidance?
Focused enough to cut noise, flexible enough to adapt to your context.

### Do I need prior experience?
Beginner friendly paths exist; some bootcamps assume basic Git + CLI.

### How are cohorts different from self-paced tracks?
Cohorts add live cadence, accountability, and peer momentum.

### Can I access replays?
Yes—events marked replay include recorded link and resources.

### How do I pick a first project?
Small scope, deployable, touches core workflow (repo → pipeline → deploy → observe).
